Веб приложение для онлайн бронирования и управления
объявлениями гостиницы

# Неавторизованный пользователь:
- регистрация в приложении
- просмотр информации о гостнице, отзывах
- просмотр списка всех вариантов номеров и информации о них
- поиск предложений бронирования по заданной дате заезда и выезда и количеству гостей

# Авторизованный пользователь:
- логин, логаут
- просмотр и редактирование профиля
- возможность бронирования номера
- возможность отмены брони
- просмотр истории бронирования
- просмотр текущих броней

# Администратор:
- управление объявлениями (добавление, удаление, изменение)
- управление информации о гостинице
- управление бронями (подтверждение, отменение)
- управление отзывами (скрытие, ответ на отзыв)
- блокировать аккаунты пользователей
- выставлять объявления об акциях
